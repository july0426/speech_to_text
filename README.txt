接口：谷歌  https://speech.googleapis.com/v1/speech:recognize?fields=results&key={your api key}
请求方式：post
提交的数据：json，包括音频文件，音频格式，语种，采样率
返回结果：json数据（识别后的文本）
限制：音频长度不能超过60s，需要提供语种
费用：每月每个账号60分钟免费，超出后0.024美元/分钟，
     一个账号的赠送金可以识别12500分钟，每天50条数据，够使用8个月。
支持语言：80种，基本覆盖全球
识别准确率：测试了中文、英文、日文，准确率80%以上

注意点：
   1.音频文件单条不能超过60s。  解决方案：对音频进行分割，识别后把结果拼接在一起。
   2.需要提供语种。  解决方案：音频文件在获取保存时，把语种也进行保存，或者写在文件名字里。

需要的扩展：
    ffmpeg        音频剪辑软件
    requests      http请求库
    MySQL-python  数据库连接
    jsonpath      json数据解析库
